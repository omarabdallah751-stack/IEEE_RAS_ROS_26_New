- What do symbols like -rw-r--r-- mean?
   -rw: meaning that the owner of the folder can Read and Write in the file.txt file
   r--: meaning that the Group can only read 
   r--:meaning that the others can only read too

- What do the three parts of the permissions represent (owner, group, others)?
   Linux Separating users which deal with folders in system into 3 Categories(owner,group,others)
       owner: the user who created the file and can read or write into it
       group:a group have been shared the folder and have a permissions from the owner
       others: not the owner or a member of the group can only read from the folder
